本教程用于指导离线安装从网上下载下来的Chrome插件crx文件。

以安装 this-is-test.crx 插件为例。

从网站上下载 this-is-test.crx文件 ，将文件保存到 F:\tmp 文件夹。

F:\tmp\this-is-test.crx

将插件 this-is-test.crx 文件解压到 F:\tmp\this-is-test文件夹。

解压前

F:\tmp\this-is-test.crx

解压后

F:\tmp\this-is-test

重命名之前

F:\tmp\this-is-test\_metadata

重命名之后

F:\tmp\this-is-test\metadata

删除前

{
    "update_url": "https://clients2.google.com/service/update2/crx",
    "key":"MIIBIjANBgkq......AQAB",
    ~~~其他内容~~~
}

删除后

{
    ~~~其他内容~~~
}

chrome://extensions/

将打包之后的crx文件拖拽到扩展程序页面进行安装。

打包好之后的crx文件可能只能在当前机器上安装，换另外的机器需要中心打包安装。
